Какие данные мы собираем

W·52 — Telegram Mini App для нетворкинга. Мы собираем минимум данных:

  • Имя, должность, компания, био — то, что вы сами вводите в приложение
  • Фотография профиля (по желанию)
  • Telegram username и базовая информация от Telegram WebApp API
  • Контакты, которые вы добавили вручную или через QR-сканер
  • Заметки, задачи, теги

AI-обработка (опционально)

Если используете голосовые заметки / AI-резюме — текст отправляется в Anthropic (Claude API) для обработки. Anthropic не использует эти данные для обучения.

Где хранятся данные

Все данные хранятся локально на вашем устройстве (localStorage). Мы не храним их на наших серверах. При очистке Telegram / браузера — данные удаляются безвозвратно. Рекомендуем экспортировать в CSV.

Передача третьим лицам

Мы не передаём ваши данные третьим лицам, кроме Anthropic для AI-обработки конкретного запроса.

Ваши права

Вы можете экспортировать данные в CSV, удалить их через настройки браузера / Telegram, прекратить использование приложения в любой момент.
